Freelance Prompt Assistant

Privacy Policy

Last updated: May 15, 2026

Overview

This extension is a local-first productivity tool designed to help freelancers organize job and profile information into structured prompts for use with external AI writing tools.

All processing occurs locally within the user's browser. The extension does not operate backend servers, cloud databases, or account systems.

Information Accessed

The extension may access and process visible information from supported Upwork pages selected by the user, including:

  • Freelancer profile information
  • Job descriptions
  • Skills and job requirements
  • User-created prompt templates and settings

Local Storage

The extension uses chrome.storage.local to save:

  • User preferences
  • Prompt templates
  • Cached profile information
  • Generated prompt history

All stored data remains on the user's device and is not transmitted to external servers by the extension.

Data Sharing

This extension does not:

  • Sell user data
  • Share user data with third parties
  • Use analytics or advertising trackers
  • Transmit user data to external servers

External AI Services

Users may optionally copy generated prompts into external AI services such as Claude, ChatGPT, DeepSeek, or Kimi.

Any data shared with those services is governed by their respective privacy policies.

Permissions Usage

The extension requests limited Chrome permissions necessary for its functionality, including:

  • storage
  • activeTab
  • scripting
  • sidePanel
  • Host permissions for https://www.upwork.com/*

Contact

For questions regarding this Privacy Policy, please contact the extension developer through the repository or distribution platform where this extension is published.
