Local-First AI Prompt Assistant for Freelancers

A Chrome side-panel extension that extracts job and profile context from supported freelance platforms and assembles structured prompts for use with external AI tools.

What It Does

  • Reads visible job information from supported Upwork pages
  • Extracts freelancer profile context locally
  • Builds structured prompts for AI-assisted proposal drafting
  • Stores all data locally using chrome.storage.local
  • Works without accounts or backend servers

Notice

Currently this extension only supports the Upwork platform. Support for more platforms coming soon.
